# FinalProject
1. პროექტის აღწერა
Guitar Website არის სტატიკური ვებსაიტი, რომელიც შექმნილია HTML-ისა და CSS-ის გამოყენებით. პროექტის მიზანია გიტარების კატალოგის წარმოდგენა, ნავიგაციის პრაქტიკა და რესპონსიული დიზაინის პრინციპების გამოყენება სხვადასხვა მოწყობილობაზე სწორად გამოსახვისთვის.
________________________________________
2. პროექტის ფუნქციონალი და გვერდები
ვებსაიტი შედგება რამდენიმე ძირითადი გვერდისგან:
•	Home
მთავარი გვერდი, სადაც განთავსებულია header, ნავიგაცია და მთავარი ბანერის სექცია. Header-ში არის დალინკვადი ლოგო და სანავიგაციო მენიუ. ელემენტების განლაგებისთვის გამოყენებულია Flexbox.
•	Catalog
გიტარების კატალოგის გვერდი, სადაც წარმოდგენილია პროდუქტები ბარათების (card) სახით. თითოეული ბარათი შეიცავს სურათს, დასახელებას და დამატებით ინფორმაციას. განლაგება შესრულებულია Grid/Flexbox-ის გამოყენებით.
•	Contact
კონტაქტის გვერდი, რომელიც შეიცავს საკონტაქტო ინფორმაციას და ვიზუალურად გამოკვეთილ სექციას. დიზაინი შეესაბამება საერთო სტილს.
